# Randomized Quiz Generator

Randomized Quiz Generator is a web-based application that generates questions from different categories such as GK, Science, etc. 
The application reads the questions aloud using the Web Speech API. After each question, a timer is set. 
The faster the user answers, the higher the score. Finally, a scoreboard is displayed showing the total marks obtained.

## Technologies Used

- **Frontend:**
  - HTML
  - CSS
  - JavaScript (Web Speech API)

- **Backend:**
  - Python (Flask)

- **Visualization:**
  - Matplotlib

- **Data Source:**
  - CSV-based Question Bank
